Real-Time Flow (SOS Emergency)

  1. User taps SOS â†’ POST /api/emergency
  2. Backend geo-queries nearest available ambulance & hospital
  3. Creates EmergencyRequest document, assigns ambulance
  4. Emits new_emergency Socket.io event to all connected clients
  5. Frontend redirects to /track-ambulance/[id]
  6. Live location updates via ambulance_location socket events
